# Clojure Challenge
Learning project for clojure

## Commits convention
Commit types

📃: Documentation
🐛: Bug fix
🎨: Style
🔁: Refactor
🔧: Config

Commit scopes

🔌: Backend
🖥️: Frontend

Commit structure

<type>(<scope>): <subject>
Example commit

```
📃: Add commit conventions to README.md
🐛(🔌): Fix bug in the backend
```
