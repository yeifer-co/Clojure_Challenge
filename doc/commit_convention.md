## Commits convention
Commit types

- 📃: Documentation
- 🌟: Feature
- 🐞: Bug fix
- 🎨: Style
- 🔁: Refactor
- 🔧: Config

Commit scopes

- Project (📦)
- Codebase (🧩)
- Testing (🧐)

Commit structure

```
<type>(<[optional]scope>): <subject>
```

Example commit

```
📃: Add commit conventions to README.md
🐞(🧩): Fix bug in the codebase
```
